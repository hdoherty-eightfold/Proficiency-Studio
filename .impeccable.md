# Design Context - ProfStudio Desktop

## Target Audience
HR professionals, talent managers, and organizational development specialists
who manage employee skills proficiency assessments at enterprise scale.

## Use Cases
- Upload employee/skills data (CSV or API integration)
- Extract skills from job descriptions using AI
- Configure proficiency levels and assessment criteria
- Run AI-powered proficiency assessments
- Review results, compare across teams, export reports

## Brand Personality / Tone
- **Brand**: Eightfold.ai - enterprise talent intelligence platform
- **Tone**: Professional, tech-forward, trustworthy, modern
- **Feel**: Clean UI surfaces with bold teal/navy branding. Confident but not flashy.
- **Primary Palette**: Teal (#88e2d2), Navy (#191841), Electric Blue (#0708ee)
- **Typography**: Plus Jakarta Sans (geometric sans-serif, similar to Gilroy)
- **Aesthetic**: Enterprise SaaS - refined, purposeful, data-driven
- **Dark mode**: Navy-based with teal accents, surface elevation for depth
